# Wave

Another Hexo theme based on Bootstrap v5.

# Feature

* Support Google Statistics
* Support Baidu Statistics

# Usage

```sh
$ cd HEXO_PATH/themes
$ git clone https://github.com/cocowool/wave.git
$ npm install --save  hexo-renderer-sass
$ cd ..
$ hexo s
```
